# Public Drug-Like Example Dataset

This folder contains public-safe molecule records for software demonstration.
The candidate set combines public kinase-inhibitor-like molecules with general
drug-like controls and one intentionally invalid SMILES validation row.

The reference panel contains public compounds used to demonstrate
reference-guided molecule triage, generated candidate prioritization,
fingerprint similarity, descriptor calculation, and public-database
differentiation signals.

The text evidence file contains synthetic demonstration notes. It does not
represent experimental results, database records, or professional advice.

These files are for software testing and portfolio demonstration only. They do
not provide legal or medical advice and do not establish any rights, safety,
efficacy, ownership, or clinical suitability.
